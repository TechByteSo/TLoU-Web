# Инструкции по минификации CSS и JS

## Автоматическая минификация (рекомендуется)

### Установка инструментов

```bash
npm install --save-dev clean-css-cli terser
```

### Запуск минификации

```bash
node scripts/minify.js
```

Скрипт создаст минифицированные версии всех CSS и JS файлов с расширением `.min.css` и `.min.js`.

### Обновление HTML файлов

После минификации обновите ссылки в HTML файлах:

**Было:**
```html
<link rel="stylesheet" href="styles/style.css">
<script src="scripts/menu.js"></script>
```

**Стало:**
```html
<link rel="stylesheet" href="styles/style.min.css">
<script src="scripts/menu.min.js"></script>
```

## Ручная минификация (альтернатива)

Если вы не хотите устанавливать инструменты, используйте онлайн-сервисы:

### CSS минификация
- https://cssminifier.com/
- https://www.minifier.org/

### JavaScript минификация
- https://javascript-minifier.com/
- https://www.minifier.org/

### Процесс:
1. Скопируйте содержимое файла
2. Вставьте в онлайн-инструмент
3. Скопируйте минифицированный код
4. Сохраните в файл с расширением `.min.css` или `.min.js`
5. Обновите ссылки в HTML файлах

## Рекомендации

- **Разработка:** Используйте обычные файлы (`.css`, `.js`) для удобства отладки
- **Продакшн:** Используйте минифицированные файлы (`.min.css`, `.min.js`) для уменьшения размера и ускорения загрузки

## Размер файлов

После минификации размер файлов обычно уменьшается на 30-70%, что значительно ускоряет загрузку страницы.
